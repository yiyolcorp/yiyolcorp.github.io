Player

Banner Player

Button Descriptions

1. Back

  • Returns to the banner list page.

2. Banner Settings

  • Switches to the Banner Settings page.

3. Viewport Editor

  • Switches to the Viewport Editor to edit the part to be displayed.
  • Edit the Viewport for Group Play.

4. Banner Rotation

  • Automatically switches to the next Banner.
  • After the last Banner, the first Banner plays again.
  • This can also be enabled/disabled in Global Settings.

5. GIF

  • Saves the current Banner as a GIF.
  • Cannot be used when Rotation is enabled.

6. MP4

  • Saves the current Banner as an MP4.

7. Reset Play

  • Restarts Banner playback.

Gesture Descriptions

Screen Tap

  • Displays the menu.
  • Menu automatically disappears after a period of user inactivity.

Swipe

  • Switches to previous/next Banner.

Showcase Player

Button Descriptions

1. Back

  • Returns to the banner list page.

2. Showcase Settings

  • Switches to the Showcase Settings page.

3. GIF

  • Saves the current Showcase as a GIF.

4. MP4

  • Restarts the current Showcase.

Gesture Descriptions

Screen Tap

  • Displays the menu.
  • Menu automatically disappears after a period of user inactivity.

Swipe

  • Switches to previous/next Showcase.

Viewport Editor

Description

1. Exit Edit

  • Exits the editing mode.

2. Minimap

  • Shows which part of the entire Banner is being displayed.

3. Guide Line

  • Blue lines.
  • Changes according to current scale or position.

4. Center Line

  • Red lines.
  • Indicates the center of the current Viewport.

5. Scale

  • Displays the current scale.
  • Ranges from 1.0x to 4.0x.

Gesture Descriptions

Drag

  • Moves the Viewport.

Pinch Out

  • Increases the zoom scale.

Pinch In

  • Decreases the zoom scale.
