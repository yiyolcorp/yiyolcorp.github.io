Banner Settings

Text File Reader

Enable Text File Reader

  • Activates the text file reader functionality.
  • This feature allows reading text with large fonts.

Text File Path

  • Specify the path of the text file.
  • Supports .txt files encoded in UTF-8.

Font Color

  • Select the font color.

Font Family

  • Select the font family.

Font Size

  • Select the font size.

Outline

  • Choose whether to use an outline.
  • When enabled, the following menu options are also activated:

Outline Width

  • Select the outline width.

Outline Color

  • Select the outline color.

Vertical Alignment

  • Select the vertical position of the text output.

Primary Text

Text

  • Enter the text string to display.

Font Color

  • Select the font color.

Font Family

  • Select the font family.

Font Size

  • Select the font size.

Outline

  • Choose whether to use an outline.
  • When enabled, the following menu options are also activated:

Outline Width

  • Select the outline width.

Outline Color

  • Select the outline color.

Vertical Alignment

  • Select the vertical position of the text output.

Scroll

  • Select the text movement type.

None

  • Text is fixed without movement.
  • All text is displayed on one screen.
    • To achieve this, text might be displayed smaller than the specified font size.

Swing

  • Text moves to the left and then reverses direction when it reaches the end, completing one cycle.

RightToLeft

  • Text starts from the right and disappears to the left, completing one cycle.

LeftToRight

  • Text starts from the left and disappears to the right, completing one cycle.

Scroll Duration

  • Select the duration of text movement.

Blinking

  • Select whether the text should blink.

Blinking Interval

  • Select the interval for text blinking.

Secondary Text

  • You can choose whether to use Secondary Text.
  • Other settings are identical to Primary Text.

Global Settings

Rotation

  • Set text rotation in 90-degree increments.
  • Supports 0, 90, 180, and 270 degrees.

Background Type

  • Select the type of background.

None

  • No background is displayed.

Static

  • Displays a solid color.
  • You can specify the background color.

JPEG/PNG

  • Displays a JPEG/PNG type image.

GIF

  • Displays an animated GIF format video.
  • This is a premium feature.

MP4

  • Displays an MP4 format video.
  • This is a premium feature.

Effect

  • Select various background effects.
  • Some background effects may obscure the background.
