LED Banner Master Manual

Default Layout

LED Banner Master (hereafter referred to as the App) is composed of three Tabs.

Banner

  • Displays the list of created banners.
  • The Free Plan allows up to 2 banners, while the Paid Plan allows up to 16 banners.
  • Button 1 is the Import button. You can import .ESZ files that were previously exported.
    • You need to have an empty banner slot to import.
  • Button 2 is the Add button. Tapping this adds a default banner to the end of the list.

Swipe a banner to the left to reveal the following menu

  • Button 1 is Banner Settings. Modify the banner settings.
  • Button 2 is Export. Export the banner as an .ESZ file for external sharing.
  • Button 3 is Duplicate. Add a copy of the banner to the end of the list.
  • Button 4 is Delete. Permanently deletes the banner.

Showcase

  • Displays the list of created showcases.
  • A showcase combines multiple banners to be displayed together on a single screen.
  • The Free Plan allows 1 showcase, while the Paid Plan allows up to 8 showcases.
  • Button 1 is the Import button. You can import .EMZ files that were previously exported.
    • You need to have an empty showcase slot to import.
  • Button 2 is the Add button. Tapping this adds a blank showcase to the end of the list.

Swipe a showcase to the left to reveal the following menu:

  • Button 1 is Showcase Settings. Modify the showcase settings.
  • Button 2 is Export. Export the showcase as an .EMZ file for external sharing.
  • Button 3 is Duplicate. Add a copy of the showcase to the end of the list.
  • Button 4 is Delete. Permanently deletes the showcase.

Settings

User Font Management

  • Manage your custom fonts.
  • Supports TTF and OTF formats.

Banner Rotation

  • Choose whether banners should play in a rotating loop.
  • If enabled, banners will play sequentially in order.
  • The duration for each banner is set by the Scroll Duration.

Auto Stop

  • Automatically stops Banner or Showcase playback after a specified time.
    • While a Banner or Showcase is playing, the device will not sleep.
    • This helps save battery by stopping playback after a set duration.

Vibration

  • Choose whether to enable vibration on events.

Maximum Brightness

  • The screen switches to maximum brightness during playback.
  • After playback stops, the screen returns to the previous brightness level automatically.
