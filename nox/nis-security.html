NOX 26.04 — 보안요구사항 전 항목 대응 완료 NOX 26.04 — Full Compliance with NIS Security Requirements

국가정보원 「국가용 보안요구사항 V3.0」 영상정보처리기기(NVR) 전체 10개 보안기능 카테고리 대응 완료 Complete coverage of all 10 security categories specified in the NIS National Security Requirements V3.0 for NVR devices

(주)이욜은 자사 영상녹화장치 NOX 26.04이 국가정보원 「국가용 보안요구사항 V3.0」 영상정보처리기기(NVR) 부문의 전체 10개 보안기능 카테고리에 대한 기술적 대응을 완료하였음을 알려드립니다. NOX 26.04는 공공기관 및 국가기관에서 요구하는 최고 수준의 보안 기준을 충족하도록 설계되었으며, 영상 데이터의 수집부터 저장, 전송, 관리에 이르기까지 전 구간 보안을 실현합니다. YIYOL Corp. is proud to announce that NOX 26.04 has achieved full technical compliance with all 10 security function categories of the National Intelligence Service (NIS) "National Security Requirements V3.0" for Network Video Recorders. NOX 26.04 delivers end-to-end security across the entire video data lifecycle — from capture to storage, transmission, and management.

10대 보안기능 대응 현황 10 Security Function Categories

1. 영상 보안1. Video Security

AES-256-GCM 알고리즘으로 영상 데이터를 실시간 암호화하여 저장합니다. 비인가 접근으로부터 영상의 기밀성과 무결성을 보장합니다.Real-time video encryption using AES-256-GCM, ensuring confidentiality and integrity against unauthorized access.

2. 식별 및 인증2. Identification & Authentication

JWT 기반 토큰 인증과 강력한 패스워드 정책으로 사용자를 안전하게 식별·인증합니다.Secure user identification through JWT-based token authentication and robust password policies.

3. 접근통제3. Access Control

역할 기반 접근제어(RBAC)로 사용자 역할에 따라 기능과 데이터 접근 권한을 세밀하게 관리합니다.Role-Based Access Control (RBAC) provides granular permission management based on user roles.

4. 데이터 보호4. Data Protection

DEK/KEK 2계층 키 관리와 TPM 봉인으로 저장 데이터를 보호합니다. HMAC-SHA256으로 무결성을 검증합니다.DEK/KEK dual-layer key management with TPM-sealed keys. Data integrity verified via HMAC-SHA256.

5. 자체보호5. Self-Protection

시스템 바이너리·설정 파일의 무결성을 실시간 모니터링하고 변조 시도를 즉시 탐지합니다.Real-time integrity monitoring of system binaries and configuration files with immediate tamper detection.

6. 업데이트 보호6. Update Protection

GPG 전자서명으로 정당한 업데이트만 설치되도록 보장합니다. 변조된 패키지의 설치를 원천 차단합니다.GPG digital signatures ensure only authentic, unmodified update packages are installed.

7. 세션 관리7. Session Management

사용자 세션의 생성·유지·만료를 체계적으로 관리하여 세션 탈취·재사용 공격을 방지합니다.Systematic session lifecycle management to prevent session hijacking and replay attacks.

8. 감사기록8. Audit Logging

보안 관련 이벤트를 빠짐없이 기록하고, 감사기록의 무결성을 보장하여 추적·분석이 가능합니다.Comprehensive logging of all security events with tamper-proof audit records for forensic analysis.

9. 암호 지원9. Cryptographic Support

AES-256-GCM, HMAC-SHA256, TLS 1.2/1.3 등 검증된 암호 알고리즘으로 기밀성·무결성을 보장합니다.Industry-standard algorithms including AES-256-GCM, HMAC-SHA256, and TLS 1.2/1.3.

10. 보안 취약점 관리10. Vulnerability Management

알려진 보안 취약점에 대한 체계적 관리와 패치 프로세스로 시스템을 안전하게 유지합니다.Systematic vulnerability tracking and patch management to maintain system security posture.
