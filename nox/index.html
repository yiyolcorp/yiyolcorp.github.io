귀사 브랜드의
NVR/VMS를 갖는 법

수억 원을 들여 처음부터 개발하지 않아도 됩니다. NOX의 마이크로서비스 아키텍처와 개방형 API 위에서, 귀사에 필요한 기능만 골라 귀사만의 솔루션을 완성하세요.

  • UI 완전 커스터마이징 — 로고, 컬러, 레이아웃, 메뉴 구조 모두 귀사 브랜드로
  • 기능 선택 구성 — 16개 마이크로서비스 중 필요한 것만 선택·조합
  • 워크플로우 커스텀 — 귀사의 업무 프로세스에 맞는 알림·자동화 규칙 설정
  • 외부 시스템 통합 — Webhook·개방형 API로 ERP, 출입통제, 스마트빌딩과 연동
  • AI 모델 교체·확장 — 기본 YOLO 외에 고객 특화 AI 모델 탑재 가능
  • OEM/화이트레이블 — SI사, 솔루션 업체가 자사 제품으로 출시 가능

NOX를 선택하는 이유

실제 운영 현장의 문제를 해결하도록 설계된 기능들입니다.

YOLO Vision AI 영상 분석

YOLO 기반 객체 감지(라인크로싱·침입)와 번호판 인식(LPR)을 단일 시스템에서 처리합니다. 이상 감지 즉시 자동 알림·녹화·Webhook 실행.

자동 알림 LPR 3-Tier 파이프라인

ONVIF Profile S/T/M 완전 호환

Profile S(스트리밍), T(고급 영상), M(메타데이터·이벤트·분석) 지원. 18개 서브패키지의 자체 라이브러리로 기존 IP 카메라를 교체 없이 그대로 연결.

기존 자산 활용 자동 디스커버리

마이크로서비스 아키텍처

16개 독립 서비스로 분리. 특정 서비스 장애가 전체를 멈추지 않습니다. 필요한 서비스만 업데이트해 무중단 운영을 유지합니다.

무중단 업데이트 수평 확장

멀티플랫폼 유연 배포

Docker 멀티 아키텍처(linux/amd64, linux/arm64)로 라즈베리파이 엣지부터 클라우드 데이터센터, Kubernetes 클러스터까지 인프라를 가리지 않습니다.

On-Premises Cloud / K8s Edge ARM

개방형 API + Webhook

Webhook으로 감지 이벤트를 외부 시스템에 실시간 전달. 개방형 API와 ONVIF 표준으로 CMS, 스마트빌딩, 보안 시스템과 자유롭게 통합.

01

상담 & 요구사항 파악

데모 & 기술 검증

커스터마이징 & 배포

운영 & 지속 발전
