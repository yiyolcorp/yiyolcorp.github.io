🎁 리딤코드 사용법

iOS와 안드로이드에서 간단하게 리딤코드를 사용하는 방법을 알아보세요

📱 iOS (아이폰/아이패드)

1

App Store 열고 프로필 터치

iOS 기기에서 App Store 앱을 실행하고, 오른쪽 상단의 프로필 아이콘을 터치합니다.

2

기프트 카드 또는 코드 교환

프로필 메뉴에서 "기프트 카드 또는 코드 교환" 옵션을 찾아 터치합니다.

3

리딤코드 입력

코드 입력 화면에서 받으신 리딤코드를 정확히 입력하고 "교환" 버튼을 터치합니다. 코드가 정상적으로 처리되면 크레딧이나 앱이 계정에 추가됩니다.

🤖 안드로이드

1

구글 플레이 스토어 접속

안드로이드 기기에서 구글 플레이 스토어 앱을 실행합니다. 홈 화면에서 플레이 스토어 아이콘을 찾아 터치하세요.

2

메뉴 열기

플레이 스토어에서 왼쪽 상단의 햄버거 메뉴(≡) 버튼을 터치하거나, 오른쪽 상단의 프로필 아이콘을 터치하여 메뉴를 엽니다.

3

코드 사용 선택

메뉴에서 "코드 사용" 또는 "기프트카드 및 프로모션 코드"를 찾아 터치합니다. 이 옵션은 보통 결제 관련 메뉴에서 찾을 수 있습니다.

4

리딤코드 입력

리딤코드 입력 화면에서 받으신 코드를 정확히 입력하고 "사용" 버튼을 터치합니다. 코드가 정상적으로 처리되면 크레딧이나 앱이 계정에 추가됩니다.

💡 유용한 팁

리딤코드는 대소문자를 구분하니 정확히 입력하세요. 코드에 문제가 있다면 고객센터에 문의하시기 바랍니다.
