üéØ Basic Usage

What is TouchDown Alarm? ‚ñº

TouchDown Alarm is an innovative alarm app that only stops when you reach your set destination.

Key features:

  • Location-based alarm dismissal system
  • 5-level difficulty settings
  • Gamified motivation system
  • Detailed statistics and analysis
How do I set up an alarm? ‚ñº

Setting up an alarm is very simple:

  • Open the app and tap 'New Alarm'
  • Set your desired time
  • Select destination location (directly on the map)
  • Choose difficulty level (Very Easy ~ Very Hard)
  • Select alarm sound
  • Tap 'Save' to complete
How does the difficulty system work? ‚ñº

The 5-level difficulty system lets you choose the intensity that suits you:

  • Very Easy: Success +10pts / Fail -5pts
  • Easy: Success +20pts / Fail -10pts
  • Normal: Success +30pts / Fail -20pts
  • Hard: Success +50pts / Fail -40pts
  • Very Hard: Success +80pts / Fail -80pts

Higher difficulty levels earn you more points.

üìç Location Services

Do I need to allow location permissions? ‚ñº
‚ö†Ô∏è Important: Location permissions are essential for TouchDown Alarm's core functionality.

Why location permissions are needed:

  • To determine your current location for destination arrival verification
  • To judge alarm dismissal conditions
  • For accurate distance calculations

Location data is used only for alarm functionality and is deleted immediately after the collection purpose is achieved.

What should I do when GPS is inaccurate? ‚ñº

Ways to improve GPS accuracy:

  • Enable Wi-Fi and Bluetooth (improves location accuracy)
  • Use outdoors (better GPS signal reception)
  • Set location services to 'High accuracy'
  • Set app location permission to 'Allow all the time'
üì± Tip: GPS accuracy may be poor indoors or underground. Try to set outdoor destinations when possible.
How is the distance to destination calculated? ‚ñº

Distance calculation method:

  • Straight-line distance calculation based on GPS coordinates
  • Real-time location updates
  • Alarm dismissal when within destination radius

The default arrival radius is 50m, which can be adjusted in settings.

‚è∞ Alarm Settings

My alarm isn't ringing. What should I do? ‚ñº

Check these items if your alarm isn't ringing:

  • Ensure device volume is sufficiently high
  • Check that silent mode is disabled
  • Verify app notification permissions are enabled
  • Exclude app from battery optimization
  • Allow background app refresh
üîã Important: Some Android devices may restrict background apps to save battery. Please exclude TouchDown Alarm in your settings.
Can I change the alarm sound? ‚ñº

We provide various alarm sounds:

  • Default alarm sounds (Marimba, Mozart, Nokia, etc.)
  • Special alarm sounds (One Piece, Star Wars themes)
  • Device's default alarm sounds available

You can change this when setting alarms or in the settings menu.

Can I set repeating alarms? ‚ñº

Yes, we provide various repeat options:

  • Daily repeat
  • Weekdays only (Mon-Fri)
  • Weekends only (Sat-Sun)
  • Select specific days
  • One-time alarm

Simply select the repeat option when setting your alarm.

üîß Troubleshooting

The app frequently crashes or runs slowly. ‚ñº

Ways to improve app performance:

  • Completely close and restart the app
  • Restart your device
  • Check for app updates
  • Free up device storage space
  • Close other apps running in the background

If problems persist, please contact customer support.

The alarm won't turn off even though I've reached my destination. ‚ñº

Check these cases:

  • Weak GPS signal areas (indoors, underground)
  • Verify destination setting is accurate
  • Check arrival radius setting (adjustable in settings)
  • Confirm location services are enabled
üí° Solution: Try increasing the arrival radius to 100m in settings, or wait briefly outdoors.
My data disappeared. Can it be recovered? ‚ñº

Data recovery methods:

  • Check auto-sync after restarting the app
  • Verify Google/Apple account login status
  • Restore from cloud backup

How do I cancel my subscription? ‚ñº

After cancellation, you can still use premium features until the end of the current billing cycle.
